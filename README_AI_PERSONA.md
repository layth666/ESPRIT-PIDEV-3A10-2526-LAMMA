# User Persona Clustering System - Documentation PIDEV Advanced

Ce module transforme votre application Symfony en un écosystème intelligent capable de comprendre l'identité comportementale de ses utilisateurs par **Clustering Non-Supervisé Simulé**.

## 🧠 Le Moteur d'Intelligence
Le `UserPersonaClusteringService` effectue une analyse transversale (Cross-Module Analysis) en temps réel :

1.  **Boutique (Logistique)** : Analyse des achats et de l'équipement possédé.
2.  **Blog (Social)** : Analyse de l'engagement (articles créés, commentaires).
3.  **Événements (Aventure)** : Analyse du volume et du type de participations.
4.  **Restauration (Gastronomie)** : Analyse des préférences culinaires via les favoris.

### Les Personas Identifiés
- **L’Aventurier Sportif** : Focus sur les événements et le terrain.
- **Le Gourmet Familial** : Focus sur les menus et les restaurants.
- **Le Bénévole Logistique** : Focus sur le matériel et la boutique.
- **L’Explorateur Culturel** : Focus sur le savoir et le blog.
- **Le Technophile Social** : Profil polyvalent et connecté.

## 🚀 Impact sur l'Expérience Utilisateur (UX Adaptative)
- **UI Dynamique** : Le Dashboard change de couleur thématique et d'icône selon le Persona.
- **Recommandations Ciblées** : Un nouveau panneau "Suggestions IA" propose des actions spécifiques au profil.
- **Navbar Intelligence** : Un indicateur "Cerveau IA" dans le menu utilisateur confirme que l'IA est active.

## 🎓 Argumentaire pour la Soutenance

> "Pour porter l'intelligence du projet à un niveau supérieur, j'ai implémenté un système de **Clustering 360°**. Contrairement à une simple base de données statique, mon application analyse continuellement les interactions de l'utilisateur à travers tous les modules pour lui attribuer un **Persona comportemental**. Cette segmentation intelligente permet de personnaliser dynamiquement l'interface et de proposer des recommandations cross-modules ultra-pertinentes. C'est une démonstration concrète de la puissance de la logique métier couplée à des concepts de Data Science appliqués, le tout intégré de manière fluide dans une architecture Symfony modulaire."

## ✅ Résultat Attendu
1.  Connectez-vous à votre compte.
2.  Allez sur votre **Dashboard**.
3.  L'IA analyse vos participations, vos achats et vos favoris.
4.  Votre **Persona** s'affiche avec un style premium.
5.  Des suggestions spécifiques apparaissent (ex: "Missions Scout" pour l'Aventurier).
6.  Naviguez sur le site : un badge IA dans la barre de navigation vous suit partout.
